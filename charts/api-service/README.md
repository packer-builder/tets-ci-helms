## Installation

```bash
helm repo update

```

## Usage Examples

### Basic Installation

```bash
helm install my-api tets-ci-helms/api-service
```

### Custom Configuration

```bash
helm install my-api tets-ci-helms/api-service \
  --set replicaCount=5 \
  --set service.port=8080 \
  --set image.tag=v1.2.0
```

### Using High Resource Profile

```bash
helm install my-api tets-ci-helms/api-service \
  --set resourceProfile=high
```

### With Custom Values File

```yaml
# my-values.yaml
replicaCount: 3
image:
  repository: my-registry/my-api
  tag: "v2.0.0"
service:
  type: LoadBalancer
  port: 80
resourceProfile: high
```

```bash
helm install my-api tets-ci-helms/api-service -f my-values.yaml
```
