## Installation

```bash
helm repo add tets-ci-helms https://packer-builder.github.io/tets-ci-helms
helm repo update
helm install my-frontend tets-ci-helms/frontend-service
```

## Usage Examples

### Basic Installation

```bash
helm install my-frontend tets-ci-helms/frontend-service
```

### Custom Configuration

```bash
helm install my-frontend tets-ci-helms/frontend-service \
  --set replicaCount=3 \
  --set service.port=3000 \
  --set image.tag=v1.0.0
```

### Expose via LoadBalancer

```bash
helm install my-frontend tets-ci-helms/frontend-service \
  --set service.type=LoadBalancer
```

### With Custom Values File

```yaml
# my-values.yaml
replicaCount: 2
image:
  repository: my-registry/my-frontend
  tag: "v1.5.0"
service:
  type: LoadBalancer
  port: 80
resources:
  limits:
    cpu: 200m
    memory: 256Mi
```

```bash
helm install my-frontend tets-ci-helms/frontend-service -f my-values.yaml
```
